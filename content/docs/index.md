# Documentação

Bem-vindo à documentação do **Tucuxi-Blast**.

## Sobre

Sistema de Record Linkage e Gestão de Dados para unificação de registros.

## Funcionalidades

- Busca inteligente com algoritmo fonético
- Unificação de registros duplicados
- Sistema de revisão com auditoria
- Processamento em lote (CSV/Excel)
